## xcode 非appstore下载安装

https://developer.apple.com/download/all/?q=xcode%2012.3

搜索你需要的xcode 版本下载，

解压下载文件 xcodeXXX.xip ,解压需要比较长时间,打开终端运行如下命令s

xip -x xcodeXXX.xip

移动文件到应用程序
